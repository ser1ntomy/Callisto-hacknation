# Callisto Skill Sense+ — AI Talent Analyzer

**Callisto Skill Sense+** is an AI-powered résumé and talent analysis tool built with **HTML**, **Tailwind CSS**, and **Chart.js**, using the **Google Gemini API** for intelligent skill analysis and visualization.  

It helps you extract, combine, and interpret professional skills and emotional traits directly from a résumé or LinkedIn summary — all within a clean, animated interface.

---

##  Features

###  AI Skill Intelligence
Each button triggers a unique AI-powered analysis:
| Feature | Description |
|----------|--------------|
| **Analyze Skills** | Extracts and lists all technical, leadership, and soft skills. |
| **SkillFusion** | Merges related skills into creative hybrid or cross-functional capabilities. |
| **SkillMirror** | Predicts 2–3 year career growth paths based on skills. |
| **SkillMatch** | Suggests relevant job titles that align with the résumé. |
| **SkillSpark** | Generates creative micro-insights about the person’s profile. |
| **SkillMood** | Visualizes confidence, creativity, growth, and leadership as a colorful doughnut chart. |
